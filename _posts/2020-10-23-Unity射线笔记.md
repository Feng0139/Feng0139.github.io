# 射线
    由一点射出，无限延申，无终点。

## 实际代码
```
Ray ray = new Ray(transform.position, transform.forward);
```
`transform.position` 起始点  
`transform.forward` 射线方向

### Debug方法
```
Debug.DrawRay(ray.origin, ray.direction * 10, Color.red);
```
